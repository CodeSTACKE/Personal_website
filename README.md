# Personal Website - Jyoti Das Portfolio

## Overview

This is a personal portfolio website for Jyoti Das, a versatile Software Developer specializing in AI-driven applications, enterprise systems, and modern web development. The site showcases professional experience, technical expertise, and featured projects.

## Features

- **Professional Profile**: Introduction and availability status
- **Technical Expertise**: Highlighted skills in Python, Java, React, and AI/LLM integration
- **Featured Projects**: Showcase of recent work including AI-integrated web applications and data operations
- **Contact Form**: Direct communication via embedded form
- **Responsive Design**: Built with Tailwind CSS for mobile-friendly experience

## Technologies Used

- **HTML5**: Structure and content
- **Tailwind CSS**: Styling and responsive design
- **JavaScript**: Interactive elements (minimal)
- **Formspree**: Contact form handling

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)

### Viewing the Site

1. Clone or download the repository
2. Open `index.html` in your web browser
3. The site will load locally without needing a server

### Development

To modify the site:

1. Edit `index.html` directly
2. For styling changes, update the Tailwind classes or add custom CSS
3. Test changes by refreshing the browser
